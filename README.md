# ⚙️ Automações de Processos com Make (Integromat)

Repositório dedicado ao armazenamento e documentação de fluxos de automação inteligentes desenvolvidos na plataforma **Make**. O objetivo destes projetos é otimizar rotinas de trabalho, eliminar tarefas repetitivas e integrar ferramentas de mercado.

---

## 🚀 Automação: Criação de campanhas de Email Marketing para o RD Station

### 📝 Descrição do Cenário
O fluxo captura na planilha os dados de modelo da campanha, link do site, o idioma e pais que será gerada a campanha e envia os dados parao Gemini Ai gere a campanha com o assunto, body em html e UTM modificada. Depois os dados ficam na aba seguinte da planilha mostrando os resultados

### 🛠️ Ferramentas & Tecnologias Utilizadas
* **Plataforma de Integração:** Make (Integromat)
* **Ações (Actions):** Google Sheets, RD Station
* **Formatos de Dados:** JSON / APIs REST

### 📥 Como Importar este Cenário
1. Faça o download do arquivo `.json` presente neste repositório.
2. Acesse a sua conta no **Make**.
3. Crie um novo cenário, clique nos três pontinhos `(...)` no menu inferior e selecione **Import Blueprint**.
4. Selecione o arquivo baixado e mapeie as suas próprias conexões.
