### 📥 Como Importar este Cenário

2. Acesse a sua conta no **Make**.
3. Crie um novo cenário, clique nos três pontinhos `(...)` no menu inferior e selecione **Import Blueprint**.
4. Selecione o arquivo baixado e mapeie as suas próprias conexões.
